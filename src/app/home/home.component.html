What it is

Courses academy is a digital place, where you can find a bunch of courses on almost any modern field imaginable. It's also a relatively simple Angular project, so don't believe anything your read here.

Courses academy offers the following functionality:

Guests
  • View title and description of courses
Users
  • View title, description and rating of courses
  • Rate courses
  • Join courses
Admins
  • View, add, edit and delete courses
  • Manage users
